Гидравлический расчет водопроводных труб

Расчетные формулы и структура

Для гидравлического расчёта водопроводных труб обычно используют формулу:
i = λ · 1/diameter_m · velocity2/2g (1)
i_1000 = λ · 1/diameter_mm · velocity2/2g
где
  • i - гидравличский уклон, м/м;
  • i_1000 - гидравличский уклон, мм/м;
  • λ - коэффициент сопротивления трения по длине;
  • diameter_m - расчетный внутренний диамер трубы, м;
  • diameter_mm — диаметр трубы, мм;
  • velocity - скорость движения воды, м/с;
  • g - ускорение свободного падения, м/с2;

Для использования формулы (1) нобходимо установить зависимости для определения коэффициента λ.

Формулы расчёта параметра i в зависимости от типа трубы:

1. Стальные и чугунные трубы (неновые)

Для неновых труб (steel_used, cast_iron_used)

Если velocity ≥ 1.2:
i = 1000 * 0.00107 * (velocity**2 / (diameter_mm/1000) ** 1.3) (6)

Где:

  • diameter_mm — диаметр трубы, мм;
  • коэффициент 0.00107 — константа из формулы;

Если velocity < 1.2:
i = 1000 * 0.000912 * ((velocity**2 / (diameter_mm/1000) ** 1.3)) * pow(1 + (0.867/velocity), 0.3) (7)

Где:

  • добавляется множитель pow(1 + (0.867/velocity), 0.3) для учёта низкой скорости потока;
  • коэффициент 0.000912 — константа из формулы.

2. Стальные трубы (новые)

В результате исследований, проведённых во ВНИИ ВОДГЕО, получены следующие зависимости для коэффициента λ:
λ = 0.312 / dp0.226 × (1.9 × 10–6 + velocity/υ)0.226(2)
lambda = 0.312 / (diameter_m ** 0.226) * ((1.9e-6 + velocity / nu) ** 0.226)
где velocity — кинематический коэффициент вязкости воды, м²/с.

Для гидравлического расчёта водопроводных труб с достаточной для практических целей точностью можно принять: velocity = 1.3 × 10–6 м²/с, что соответствует температуре воды 10 °C.
При этом значении velocity формуле (2) можно придать вид:
λ = 0.0159 / dp0.226 × (1 + 0.684/velocity)0.226(2a)
lambda = 0.0159 / (diameter_m ** 0.226) * ((1 + 0.684 / velocity) ** 0.226)

2. Чугунные трубы (новые)

λ = 0.863 / dp0.284 × (0.55 × 10–6 + velocity/velocity)0.284 (3)
lambda = 0.863 / (diameter_m ** 0.284) * ((0.55e-6 + velocity / velocity) ** 0.284)

Или, приняв velocity = 1.3 × 10–6 м²/с:

λ = 0.0144 / dp0.284 × (1 + 2.36/velocity)0.284(3a)
lambda = 0.0144 / (diameter_m ** 0.284) * ((1 + 2.36 / velocity) ** 0.284)
