Градусо-сутки отопительного периода (ГСОП)

Градусо-сутки отопительного периода, (°C·сут/год), определяют по формуле:

ГСОП = (tв – tот) · zот (5.2)

где:

  • tв — расчётная температура внутреннего воздуха здания (°C), принимаемая при расчёте ограждающих конструкций групп зданий, указанных в таблице 3:
    • по поз. 1 — по минимальным значениям оптимальной температуры соответствующих зданий по ГОСТ 30494;
    • по поз. 2 — согласно классификации помещений и минимальных значений оптимальной температуры по ГОСТ 30494;
    • по поз. 3 — по нормам проектирования соответствующих зданий;
  • tот — средняя температура наружного воздуха (°C) и продолжительность (сут/год) отопительного периода соответственно, принимаемые по среднесуточной температуре наружного воздуха не более 8 °C;
  • zот — продолжительность отопительного периода (сут/год).

При проектировании дошкольных образовательных организаций, общеобразовательных организаций, медицинских организаций и домов-интернатов для престарелых — не более 10 °C.

Примечания

  1. Значения для величин ГСОП, отличающихся от табличных, следует определять по формуле

    Ro = a · ГСОП + b,

    где

    • ГСОП - градусо-сутки отопительного периода, (°C·сут)/год, для конкретного пункта
    • a, b - коэффициенты, значения которых следует принимать по данным настоящей таблицы для соответствующих категорий зданий.

    Для графы 6 для интервала свыше 12000 (°C·сут)/год следует принимать базовое значение требуемого сопротивления теплопередаче равным значению для 12000 (°C·сут)/год.

  2. Для зданий с избытками явной теплоты более 23 Вт/м3 нормируемые значения приведенного сопротивления теплопередаче должны определяться для каждого конкретного здания.
