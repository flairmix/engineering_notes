Комплексное требование: расчёт удельной теплозащитной характеристики здания

Нормируемое значение удельной теплозащитной характеристики здания kтроб (Вт/(м³·°C)) определяется в зависимости от:

  • отапливаемого объёма здания;
  • градусо-суток отопительного периода (ГСОП) района строительства.

Данные берутся из таблицы 7 с учётом примечаний.

Примечания

  1. Для промежуточных значений объёма здания и ГСОП, а также для зданий с отапливаемым объёмом > 200 000 м³ значение kтроб рассчитывается по формулам:
    • При Vот ≤ 960:

      kтроб = (4.74 / (0.00013 · ГСОП + 0.61)) · (1 / ∛Vот) (5.5)

    • При Vот > 960:

      kтроб = 0.16 + (10 / √Vот) / (0.00013 · ГСОП + 0.61) (5.5)

    (формула 5.5)

  2. Если значение kтроб, рассчитанное по формуле 5.5, меньше значений рассчитанных по формуле 5.6, следует использовать формулу 5.6:

    kтроб = 8,5 / √ГСОП (5.6)

Таким образом, для корректного расчёта удельной теплозащитной характеристики здания необходимо использовать таблицу 7, формулы (5.5) и (5.6) с учётом примечаний.
